\begin{abstract}
This thesis describes a web application IT Tech for gathering and sharing information in the form of articles about software development. The application also includes expanded systems of leaving feedback and article rewiew. In the introduction of this thesis there is a brief about the need to implement the system. Then there are analysis and evaluation about exsiting similar services and later are proposed mechanisms that could solve the biggest problems of this services. After introduction there is project of the system in which are described functional and non-functional requirements. Based on requirements and introduction, was specified system architecture. IT Tech is distributed web application with REST as a main form of communication. In system architecture there are client app, server app, security layer and SQL and NoSQL databases. Then was specified use cases with their details and designed user interface prototypes according to them. Last part of the design was create ERD diagram for SQL database and schemas for NoSQL database. The next step is implementation of system. Server app was implemented mainly using Spring framework and React library was used for implementation of client app. Last step of system implementation was introduction of security layer in form of an external server for authentication and authorization called Keycloak. Backend app was verified by using integration tests of controllers. The next step is the described deployment of the entire system using Azure and MongoDB Cloud platforms. At the end of the thesis, there is a deployment instruction which describing local start of the system.
\end{abstract}
